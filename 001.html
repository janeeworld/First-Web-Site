Hypertext Markup Language (HTML)

The standard markup language for creating web pages and web applications.
Web browsers receice HTML documents from a web server or from local storage and render them into multimedia web pages.
HTML describes the structure of a web page semantically and originally included cues for the appearance of the documents.

HTML elements are the building blocks of HTML pages. With HTML consturcts, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured docutments by denoting structural semantics for text such as headings, papragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
